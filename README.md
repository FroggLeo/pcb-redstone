# PCB Redstone
Minecraft redstone but with real PCB tiles, connecting magnetically and functions just like the game!

![Picture of the redstone dust](https://hc-cdn.hel1.your-objectstorage.com/s/v3/1818c1a86367f81e9f6072ebd77a43ea151e5317_image.png)

## Features:
- RP2040 microcontroller
- 40*40mm compact design 
- Dual 4 pin magentic pogo pins to connect on any side (2 pins per channel)
- Lots of toggleable LEDs to match block functions!
- Simulate Java or Bedrock redstone mechanics
- Program with 2.54mm holes using SWD
- Communicate with 2 GPIO inputs/outputs
- Inject 12v sources to any tiles and power all the blocks
